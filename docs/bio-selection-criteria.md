# 바이오주 선별 기준 (유튜브 정리)

> 출처: [바이오주식 잘 고르는 법](https://www.youtube.com/watch?v=d2edRs5xZhQ)

바이오는 **가장 난이도 높은 섹터** 중 하나이며, 아래 원칙들을 종합적으로 체크해야 한다.

---

## 1. 특허권에 주목하라

- **특허는 기본 중의 기본** — 특허가 없으면 독립적 사업 자체가 불가
- 특허 요건: 신규성, 진보성, 산업상 이용 가능성
- **단, 특허가 있다고 과신 금지** — 특허는 "다르다"는 증명이지 "뛰어나다"는 보장이 아님
- 아마추어가 특허의 질적 수준을 판단하기 어려우므로, **전문가의 평가**를 봐야 함:
  - 애널리스트 리포트에서 극찬이 있는가?
  - **글로벌 빅파마가 큰 돈을 내고 사려 하는가?**

### 위험 신호 (Red Flags)
- 특허 자체가 없거나, 출원조차 안 됨
- 특허 심사에서 거절당함
- 특허가 있다고 주장하지만 전문가 칭찬 없음
- 글로벌 빅파마가 관심을 보이지 않음

---

## 2. 논문에 주목하라

- 특허 출원 후 → 논문 발표가 자연스러운 순서 (특허 전 논문 공개 시 신규성 훼손 위험)
- 논문 = **학계의 검증을 요청하는 시그널**
- **피어 리뷰(동료심사) 통과 여부**가 핵심:
  - 데이터 조사 방법론, 데이터양, 결과의 신뢰성, 통계 해석의 정확성을 학계가 인정했다는 의미

### 체크 포인트
| 항목 | 좋은 신호 | 위험 신호 |
|------|-----------|-----------|
| 논문 존재 | 있음 | 없음 → 꽝 |
| 게재 저널 | 유명·권위 있는 저널 | 듣보잡 저널 / 돈 주면 실어주는 저널 |
| 피어 리뷰 | 통과 | 비난받음 / 거절 |
| 인용 수 | 많음 (재인용 활발) | 적음 → 꽝 |

---

## 3. 주요 학회 초청 발표에 주목하라

- 학회 참가 ≠ 학회 발표 — **이 구분이 매우 중요**

### 포스터 발표 (가치 낮음)
- 포스터/배너 하나 걸어놓고 전단지 배포, 사진 촬영하는 수준
- "대학회 성공적 발표, 세계적 주목" 등의 홍보 = **대부분 뻥**
- 비행기값만 있으면 누구나 갈 수 있음

### 구두 초청 발표 (진짜 가치)
- 학회가 **공식 초청**하여 발표를 시키는 것
- 헤드라이너, 주제 발표, 연사 자격 부여 = **까다로운 사전 심사 통과 의미**
- 전 세계 권위자들 앞에서 기술을 제대로 발표한 이력

### 학회의 권위도 중요
- ASCO, AICR, ASH 등 **세계적으로 유명한 학회**인지 확인
- 이름만 거창하고 실제로는 무명 학회인 경우 주의

---

## 4. 임상 승인과 데이터 발표에 주목하라

- 임상 없이는 이론·상상에 불과 — **실증 데이터가 필수**

### 임상 단계
1. **전임상**: 세포·동물 실험
2. **임상 1상**: 안전성 (죽나 안 죽나, 큰 부작용 없나)
3. **임상 2상**: 유효성 (기존 약 대비 얼마나 효과적인가)
4. **임상 3상**: 대규모 임상, 종합 체크
5. **허가 신청**: 모든 단계 완료 후 관계 당국에 허가 요청

### 위험 신호 (Red Flags)
- 임상 데이터 없이 **"혁신적 플랫폼", "게임 체인저", "글로벌 최초", "판도가 뒤집힌다"** 등 화려한 수식어만 나열 → 숫자 없는 홍보는 뻥
- **"임상 신청"과 "임상 승인"을 혼동시키는** 경우 → 신청은 누구나 가능
- 승인은 받았지만 **결과치를 숨기는** 경우

### 긍정 신호
- **임상 2상 이상 진행 중**이면 의미 부여 가능 — 중요 단계를 넘긴 것
- 각 단계별 데이터가 투명하게 공개됨
- 임상 결과가 기존 제품 대비 **압도적 효과**를 보임

## 5. 글로벌 빅파마와의 라이센스 아웃 경력

- 전문가(빅파마)가 **직접 검증하고 큰 돈을 주고 산 이력** = 가장 강력한 인증
- 기술·논문·임상을 직접 판단하기 어려울 때, 빅파마의 투자 여부가 대리 지표 역할

### 체크 포인트
| 항목 | 좋은 신호 | 위험 신호 |
|------|-----------|-----------|
| 계약 상대 | 일라이릴리, 화이자 등 **진짜 글로벌 빅파마** | 이름 못 들어본 외국 회사 = 빅 아님 |
| 계약 이력 | 꾸준히 성공, 파기 없음 | 자주 파기, 계약금 반환 이력 |
| 계약 내용 공개 | 투명하게 공개 | 내용 비공개 → 의심 |
| 계약 규모 | 실질적 | 겉만 큰데 실속 없음 |

---

## 6. 계약 세부 구조를 파악하라

### 반환 의무
- **반환 의무 없는 계약** = 자신감 있는 좋은 계약
- 반환 의무가 있으면 → 빅파마는 기술이 가짜면 다 회수 가능 → 실질적 가치 의문

### 마일스톤 (성과 기반 지급)
- "총 몇 조 계약" 이라고 해도 대부분이 마일스톤(조건부 금액)이면 → 실제 수령액은 미지수
- **제품화 단계 실패 시 중간 마일스톤 수령 불가**
- 계약 총액이 아니라 **확정 수령액과 조건**을 봐야 함

### 공시 규정 체크
- 계약이 식별 가능할 것 (무슨 계약인지 명확)
- 각 주체별 수행 의무가 식별될 것
- 거래 가격 산정이 명확할 것
- 거래 가격이 수행 의무에 따라 배분될 것
- 수행 의무 이행 후 수익 인식 가능할 것

### 홍보 태도로 신뢰 판단
- 조건부 금액을 마치 확정인 것처럼 홍보 → **허세, 신뢰 불가**
- 정직하게 조건부라 밝히고 조용히 실적 쌓는 회사 → 신뢰 가능

## 7. 경영진과 자금 구조를 보라

### 경영진 (가장 중요한 마지막 기준)
- **사장이 기술자 출신인가?** (오너셰프 비유 — 기술을 아는 사람이 경영해야 함)
- 이사회(보드) 과반이 **기술 전문가, 창립 멤버**인가?
- CTO 등 기술 총책임자가 **연구에 진심인 사람**인가?
- 처음부터 함께한 창립 멤버들이 뜻을 모아 연구에 미쳐 있는 팀인지 확인

### 위험 신호
- 경영진에 기술자가 없음
- 돈만 가져온 외부 투자자가 경영권을 장악
- 폼만 잡고 실질적 연구를 하지 않는 팀

### 자금 구조
- 바이오는 초창기 외부 투자(벤처캐피탈, 엔젤 등)가 필수적으로 들어감
- 외부 투자 자금은 **엑싯 타이밍**이 정해져 있음 (2년, 5년 등)
- 엑싯 시점이 다가오면 → **가짜 호재를 만들어 주가를 띄우고 빠지려는 유혹** 발생

### 체크 포인트
- 투자 자금의 **질**: 단기 투기 자금 vs 장기 바이오 전문 펀드
- 갑작스러운 호재가 펀드 엑싯 타이밍과 맞물리지 않는지
- 자금 출처가 투명하고, 롱텀 투자인지 확인
