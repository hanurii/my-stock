## 한 줄 정의

> **상승장에서, 상대강도 최상위(L)인 종목을, 거래량이 조용해진 눌림목

> 보수 익절·빠른 급등은 8주 보유로 분기하되(현 보수→점차 과감화),

> 외국인/기관 수급(I)은 *고를 때*가 아니라 *상승의 연료*로 본다.

## 5기둥 + 마스터 스위치

### ① 무엇을 — 선별 (한국식 CAN SLIM v1.1) ✅확정
- **L (상대강도, RS)= 압도적 핵심.** 위너를 비위너에서 가장 강하게
  가려냄(enrichment 최고), 두 사이클 재현.
- **결합 > 단독**: L + 수급(I) 결합이 단독 축보다 훨씬 강함(c2024-12
  I+L 2.89x / c2020-03 1.92x, 순서 동일).
- **C (직전분기 EPS YoY>0)** = 약한 가점.
- **M (시장 상승국면)** = 보조 + ★스위치 근거.
- **버릴 것**: S(소형·대주주·무희석)·+K(원화약세)·A(연간흑자 요구)
  = 선별 변별력 없음/역효과 → *종목 필터로 쓰지 말 것*(생존자 편향
  교정). 적자에서도 폭발(A 요구 폐기).
- **섹터 무리 (보편 보조축 — 세기 가변)**: 위너는 안오름보다 섹터
  집중(cross-cycle 위너200 vs 안오름500: c2024-12 격차+0.043 p=0.0000 /
  c2020-03 +0.009 p=0.0360 — **양쪽 다 통계적 성립**). 단 효과 크기가
  시장구조에 강의존(소수테마장 ≈5×). **결론: 보편 위너 특성이되
  *세기 가변* — L·I 다음 보조 가점 *항상 켜되 시장 집중도에 비례*
  (소수테마장 강·광범위장 약), 단독 컷 아님.** ⚠ n=100 때 p=0.10
  (비유의)은 검정력 부족 착시 — n=200서 정정. cross-cycle은 위너
  200 필수. `_sector_cluster_n500.txt`·`_sector_cluster_xcycle2020_w200.txt`.
- **중요**: 이 요건 충족 시점은 *최적 매수보다 ~3개월·+57% 늦음* →
  **선별 요건은 매수 전 필터가 아니라 *보유 후 수렴 모니터링 기준*.**
- **c2020-03 N=100 독립 재확인(2026-05-17)**: base 14일·깊이 10%·
  거래량 1배·신고가 79%·큰 선행상승 = c2024-12와 동일(골격 견고).
  C/A 무용(EPS 중앙 ~0·적자 무방)·+K 반전(원화 *강세*인데 위너,
  c2024-12는 약세 — 환율 무용 확정). RS 중앙 71(표본↑ 시 하락,
  pivot 비절대). 표본↑ 시 극단치만 둔화·방향 불변.

### ② 언제 — 매수 타이밍 ✅확정
- 보편 "마법 신호" 없음. 조건을 "짧은 시간 +20%"로 구체화하니 시그니처
  발견 = **5신호 동시**:
  1. 거래량이 평소(50일평균) **이하**(조용)
  2. **신고가 한참 아래** (52주 고가의 ~88% 이하)
  3. **50일선 근처** (±10%)
  4. **외국인 또는 기관 60일 순매수**(매집 배경)
  5. **RS ≥ 50** (상대강도 시장 중간 이상)
- = 오닐 "거래량 터지는 신고가 돌파"의 **정반대 = 조용한 눌림목 재가속**.
- 효과: 깨끗한 비무버 대비 **~3~6배**(작은 목표일수록 우위; +20~25%
  유의). **헛신호 절반 이상 = 확률 도구이지 확실 아님.** 두 강세장 재현.
- **선별 강화(#3, 2026-05-18)**: L 에 *제외필터* "큰 선행상승(≥50%)"
  결합 시 자연혼합 정밀도 76→90%·lift 7.8→23.6x(위너 57→52% 보존).
  → 권장 = L + 선행상승. `korea_canslim.md` 참조.
- **분할 vs 일괄(2026-05-18, 사후, c2024-12 *및* c2020-03 교차)**:
  **두 사이클 모두** 일괄(LUMP)이 전 그룹·지표 최고, 순위
  LUMP>SPLIT_D>SPLIT_W>SPLIT_DIP 동일(교차 견고). 혼합 H60
  c2024-12 +77.5% / c2020-03 +84.6% (분할 −16~25%p). **단 양
  사이클 다 생존자편향 → 분할 하방보험 가치 미측정·일괄우월은
  강세장·재앙제외 조건부.** 분할=기대수익 깎아 변동성·심리 사는
  보험(의도적). 분할 시 *하루단위 > 주단위 ≫ 눌림대기*. 요일효과
  **교차 비일관**(c2024-12 월≈그외 / c2020-03 월 −9%p)→무시,
  **오전장=종가데이터 검정불가**. `_split_vs_lump*.txt`.
- **선별강화 #3 임계(2026-05-18)**: L+선행상승≥**50%** 교차 권장
  (2사이클 정밀도 도약·위너잔존 허용). screen_v11 미반영(분석전용,
  통합은 승인후). 오늘 6픽 전원 선행상승≥50(143~1128%)=필터로
  탈락 없음. `_trap_filter_sweep.txt`.

### ③ 손절 ✅확정
- 오닐 **−8%는 한국에 과도** — 초기 타이트 스톱이 *미래 위너를 학살*
  (위너 그룹서 −8% 컷의 100%가 나중 +100%↑ 종목).
- **단일 −15% 재해 손절** 채택. '잘못 진입' 통제는 손절폭이 아니라
  *진입 신호 질(5신호)*; 손실 공포는 *매수 비중*으로.
- 약세장에선 정반대(타이트가 덜 잃음) → 손절폭도 국면 의존.

### ④ 매도 ✅방향 확정
- 오닐 "+20% 즉시 매도"는 한국 큰 흐름엔 독: HMM·데브시스터즈 실증
  **+56% vs +926%/+1,460%**. **넓은 트레일링이 압도.**
- 50/120일선 이탈 매도 = 한국 급등주엔 과민 → **폐기.**

### ⑤ 보유·수익 — 보수 익절 + 8주 예외 ✅운영확정 (2026-05-18 v2, 사용자)
"끝까지=최대수익" 거부 유지. 매수 평단 기준, **속도로 분기**:

- **토대(불변·최우선)**: 평단 **−15% 재해 손절**(항상 살아있는 바닥)
  · **★코스피 200일선 하향 → 전량 현금**.

- **분기 A — 빠른 급등(평단 확정 후 15거래일≈3주 내 +20% 도달)**
  = O'Neil 폭발 위너 예외, **8주(40거래일) 보유 원칙 ON**:
  - 8주 동안 아래 보수 사다리·트레일링 *적용 안 함*. −15% 손절·
    ★스위치만 유효(빨리 자르지 마라).
  - **4주(20거래일) 중간 점검(재량 매도 *검토*, 강제 아님)**: ①
    지지부진(횡보·신고가 동력 상실) ② 큰손 이탈(외인·기관 60일
    순매수 → 순매도 전환) ③ 더 매력적 종목 발견 — 셋 중 하나면
    8주 전이라도 매도 검토.
  - **8주 경과 → 재판단**: 추세·강세 만족 → 최대 6개월(126거래일)
    연장 보유 적합성 검토(전용 데이터 확인=열린항목). 불만족(8주
    룰 기대 미달) → 8주 원칙 해제, 매도 또는 분기 B 복귀.

- **분기 B — 느린/일반 상승(분기 A 트리거 미충족)** = 보수 익절:
  1. **+20% 도달 → 보유분 1/3 익절**
  2. **+25% 도달 → 추가 1/3 익절**
  3. **잔여 1/3 → 고점(종가 최고) 대비 −5% 이탈 *또는* ★스위치
     중 먼저 → 청산**

- **진화 의도(명시)**: 분기 B는 *의도적 보수* — 고점−5%는 한국
  급등주 일변동(±5% 일상)상 사실상 "+20~25% 전량 청산 ≈ O'Neil
  즉시매도(큰 줄기 ~8%만, 데이터 비최적)". 사용자가 *현금 안정을
  눈으로 확인하며 점차 트레일링을 넓혀 과감화* 예정 → 재논의는
  사용자 판단 트리거. 분기 A가 큰 위너 상방을 보전하는 안전판.
- **한계**: 분기 B·4주 점검 기준(지지부진/더 매력적=재량)은 검증값
  아닌 합의 파라미터. 분기 A 8주→6개월 연장은 기존 데이터 부분근거만
  (6개월변형 WIN 74.2% < 트레일링 85.3% > +20%즉시 36.5%) — 전용
  재검증 차기. 남은 구멍: +20% 못 가고 페이드 시 −15%까지 손해·수익
  0(본전 보호선 미채택, 추후 옵션).

### ★ 마스터 스위치 — 약세장 감지 ✅확정 (v1-g)
- 우리 매수·출구는 **강세장 전용**(2022 약세장선 lift<1·평균 손실,
  데이터로 입증).
- **스위치 = 코스피 종가 < 200거래일 단순평균 AND 그 200일선이
  20거래일 전보다 낮음 → 보유 *전량 현금화*** (해제 = 종가가 상승
  전환한 200일선 위 복귀 시 재투입).
- 단순 이분법이 "절반만 먼저 빼는 조심" 중간단계보다 우월(검증).
- 200일선은 고점 대비 ~−10%·늦게 켜짐 — *천장 맞히기가 아니라
  이후 추가 폭락 회피*가 목적(−10%서 현금화→이후 −27% 회피).
- **재진입 ✅확정**: 거시/쇼크/원칙 매도 후 → *별도 규칙 없음.
  매수 규칙(★강세 + ②5신호)을 그대로 재적용, 다시 충족하는 첫
  종가일 재진입.* V쇼크면 수일 내 재점등(자본 복구), 진짜
  추세약세장이면 점등 안 됨→현금 유지(정답). 검증: 심텍 03-04
  −22% 매도 → 03-05 재진입 → 왕복 +56%(SELL_CASH −22% 대비).

## 핵심 통찰 (정수)

1. 돈 버는 힘은 "마법 매수 신호"가 아니라 *좋은 종목 선별(L) + 조용한
   진입(②) + 넓은 보유(④) + 약세장 회피(★)* 의 **시스템 전체**에 있다.
2. **외국인/기관(I)은 *고르는* 신호가 아니라 *큰 상승의 연료*.** 주체는
   종목·섹터·시장 의존(대형 해운=외국인, 중형 게임=기관) → "외인 OR
   기관" 결합 정의가 옳음.
3. **오닐 철학은 한국서 옳다, 숫자는 틀리다.** −8%·+20% 그대로면 한국
   위너를 죽인다.
4. **국면(M)이 master switch.** 같은 규칙이 강세장 +수익·약세장 −손실.
